## git常用指令 ##  
一些常用指令，當然指令是要實際看狀況而用  


| 指令 | 說明 | 
| --- | --- | 
| init | 初始化。告訴這個專案加入git的功能 | 
| status | 查看狀態。 |
| add | 加入版本控制的行列。新檔案要版控必加 |
| commit | 建立新的版本。 |
| log | 查看歷史紀錄。 | 
| [checkout]() | 查看指定的版本狀態。 |
